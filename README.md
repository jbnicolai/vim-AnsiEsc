# Vim AnsiEsc

A [pathogen](https://github.com/tpope/vim-pathogen) and [Vundle](https://github.com/gmarik/Vundle.vim) friendly wrapper around [Charles Campbell](http://www.vim.org/account/profile.php?user_id=96)'s [AnsiEsc](http://www.drchip.org/astronaut/vim/index.html#ANSIESC)

Convert ansi color codes to vim text coloring

### Demo

[![Demo tmux-fpp](http://g.recordit.co/BYaxhyUXzU.gif)](http://recordit.co/BYaxhyUXzU)

### Usage

- `:AnsiEsc` - toggles Ansi escape sequence highlighting
- `:AnsiEsc!` - rebuilds highlighting for new/removed three or more element Ansi escape sequences.
